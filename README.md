# 🤖 Accident Severity & Cat-Dog Classification AI

An integrated Artificial Intelligence application built with **Streamlit** that combines two different AI models into a single web application.

- 📊 **Accident Severity Prediction** using **Extreme Gradient Boosting (XGBoost)**.
- 🐱🐶 **Cat & Dog Image Classification** using **MobileNetV2 (TensorFlow/Keras)**.

---

## 📌 Overview

This project demonstrates the implementation of both **Machine Learning** and **Deep Learning** in one application.

### Project 1 — Accident Severity Prediction

Predicts accident severity levels from tabular accident data using the **XGBoost Classifier**.

### Project 2 — Cat & Dog Image Classification

Classifies uploaded images into **Cat** or **Dog** using a **MobileNetV2** Convolutional Neural Network.

---

# ✨ Features

## 📊 Accident Severity Prediction

- Interactive accident data input
- Predict accident severity
- XGBoost machine learning model
- Fast prediction results

## 🐱🐶 Cat & Dog Image Classification

- Upload image (.jpg, .jpeg, .png)
- Classify Cat or Dog
- Display confidence score
- MobileNetV2 Deep Learning model

# 🚀 Installation

## Requirements

- Python **3.12**
- pip

Check Python version

```bash
python --version
```

Expected output

```text
Python 3.12.x
```

---

## Clone Repository

```bash
git clone https://github.com/achmadwachid/accident-severity-and-catdog-classification.git

cd accident-severity-and-catdog-classification
```

---

## Create Virtual Environment

```bash
python -m venv .venv
```

---

## Activate Virtual Environment

### Windows (Git Bash)

```bash
source .venv/Scripts/activate
```

### Windows (Command Prompt)

```cmd
.venv\Scripts\activate
```

### Windows (PowerShell)

```powershell
.\.venv\Scripts\Activate.ps1
```

### Linux / macOS

```bash
source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Streamlit

```bash
streamlit run app.py
```

The application will be available at

```text
http://localhost:8501
```
